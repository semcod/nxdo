# nxdo examples

Real outputs generated by running nxdo **on the nxdo repository itself** (2026-06-16).

| File | Command | Description |
|------|---------|-------------|
| [nxdo-self-context.txt](./nxdo-self-context.txt) | `nxdo print-context . --max-commits 5 --raw` | Project snapshot + git context sent to the LLM |
| [nxdo-self-metrics.txt](./nxdo-self-metrics.txt) | `nxdo metrics . --top 3` | Complexity, coupling, hotspots (no LLM) |
| [nxdo-self-plan.json](./nxdo-self-plan.json) | `nxdo plan . -e "After renaming lane to nxdo…" --max-commits 10 --json` | Validated 10-task `TaskPlan` JSON |

## Regenerate

```bash
cd /path/to/nxdo
export OPENROUTER_API_KEY=...

nxdo print-context . --max-commits 5 --raw > examples/nxdo-self-context.txt
nxdo metrics . --top 3 > examples/nxdo-self-metrics.txt
nxdo plan . \
  -e "After renaming lane to nxdo, what are the next sensible engineering steps?" \
  --max-commits 10 --json > examples/nxdo-self-plan.json
```

## Validate the sample plan

```bash
nxdo validate examples/nxdo-self-plan.json
```
